# covid19-dashboard-vaccini

### Local Dev
Install all dependencies and start
```bash
$ yarn install
$ yarn start
```

### Local Prod
Install all dependencies and build
```bash
$ yarn install
$ yarn build
```
Then `./dist` will be ready to be served.
